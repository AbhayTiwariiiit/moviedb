# HD4U Movies Database

A modern, high-performance movie discovery platform built with Astro, Tailwind CSS, and the TMDB API. Features server-side rendering (SSR), dynamic routing, and real-time OTT streaming data.

## Features
- **Astro 5 SSR**: Blazing fast server-side rendering for optimal performance and SEO.
- **Tailwind CSS v4**: Utility-first styling with a sleek, Vercel-inspired UI.
- **TMDB API Integration**: Real-time fetching of movie metadata, cast, box office stats, and streaming availability.
- **Fully Responsive**: Perfectly scaled for mobile, tablet, and desktop viewing.

## Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Create a `.env` file and add your TMDB API Key:
   ```env
   TMDB_API_KEY=your_api_key_here
   ```

3. Start the dev server:
   ```bash
   npm run dev
   ```
